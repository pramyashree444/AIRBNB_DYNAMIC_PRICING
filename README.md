# AIRBNB_DYNAMIC_PRICING
AIRBNB_PRICING_ENGINE
Airbnb Dynamic Pricing Analysis

📌 Overview

This project analyzes Airbnb listing data and builds a simple pricing recommendation system using data analytics and machine learning.

---

🎯 Objectives

- Analyze factors affecting prices
- Perform data cleaning and visualization
- Build a price prediction model
- Create an interactive dashboard

---

📂 Dataset

- Source: Kaggle
- Dataset: NYC Airbnb Open Data

---

🛠️ Tools Used

- Python
- Microsoft Excel
- Microsoft Power BI

---

🤖 Model

Linear Regression model used to predict prices based on:

- Minimum nights
- Number of reviews
- Availability

---

📊 Dashboard

Built in Microsoft Power BI showing:

- Price by location
- Room type comparison
- Reviews vs price

---

🚀 Conclusion

This project demonstrates how data-driven techniques can be used to optimize pricing strategies on Airbnb.
